## Cheese Burst Pizza

[toc]

#### Ingredients
1. All-purpose Flour
2. Yeast
3. Salt
4. Mozzarella Cheese
5. Veggies

#### Recipe
1. Make the dough.
2. Let it rest for sometime (15-20 mins).
3. Make pizza sauce.
4. Pre-heat oven 180 degrees celsius.
5. Prepare pizza base.
6. Add veggies and cheese.
7. Bake for 15 mins.
